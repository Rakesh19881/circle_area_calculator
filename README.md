# Circle Area Calculator

A simple Python program to calculate the area and circumference of a circle, and estimate fencing costs.

## Features
- Calculate area of a circle given the radius
- Calculate circumference of a circle
- Estimate fencing cost based on circumference

## How to Run
```bash
python circle_area_calculator.py
```

## Requirements
- Python 3.x
- No external dependencies required (uses built-in math module)
